# Dijkstra Algorithm Implementation

## Description
This project implements Dijkstra’s algorithm in Python to find the shortest path in a weighted graph. It uses a priority queue for efficient computation and includes test cases for validation.

## Features
- Shortest path calculation using Dijkstra’s algorithm
- Efficient implementation using priority queue
- Modular code structure
- Test data support for validation

## Files Overview
- main.py → Entry point to run the program
- dijkstra.py → Core implementation of Dijkstra’s algorithm
- pqdict.py → Priority queue implementation
- test.py → Test cases for validation
- test_data/ → Sample input data for testing

## Technologies Used
- Python

## How to Run
1. Navigate to the project folder:
   cd project-folder

2. Run the program:
   python main.py

## Use Case
This project demonstrates graph traversal and shortest path algorithms, useful in applications like navigation systems, network routing, and optimization problems.
